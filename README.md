# VoteApp Online 🗳️

פלטפורמת הצבעה מקוונת מתקדמת לניהול בחירות דמוקרטיות ושקופות.

## 🌐 גישה לאתר
[https://voteapp.online/](https://voteapp.online/)

## 📋 תיאור הפרויקט

VoteApp היא מערכת הצבעה מקוונת מקיפה המאפשרת ארגון וניהול של תהליכי בחירות באופן דיגיטלי, יעיל ובטוח. המערכת מספקת פתרון מלא להצבעות בארגונים, קהילות, מוסדות חינוך ועוד.

## ✨ תכונות עיקריות

### 👥 ניהול קבוצות
- יצירה וניהול של קבוצות הצבעה מרובות
- הגדרת הרשאות וזכויות גישה לכל קבוצה
- ממשק ניהול אינטואיטיבי למנהלי קבוצות

### 🎯 מערכת קמפיינים
- יצירת קמפיינים בחירות מותאמים אישית
- הצגת מידע מפורט על כל קמפיין
- תזמון והגדרת תקופות הצבעה

### 🗳️ הצבעה למועמדים
- רישום והצגת מועמדים עם פרטים מלאים
- מערכת הצבעה אינטואיטיבית וידידותית למשתמש
- אבטחת תהליך ההצבעה ומניעת הצבעות כפולות

### 📊 דוחות ותוצאות
- מעקב בזמן אמת אחר תוצאות ההצבעה
- דוחות מפורטים וגרפים ויזואליים
- שקיפות מלאה בתהליך הספירה

## 🛠️ טכנולוגיות

הפרויקט נבנה באמצעות טכנולוגיות מתקדמות:

- **Frontend**: Node.js + React
- **State Management**: Redux

## 🚀 התחלת עבודה

```bash
# התקנת תלויות
npm install

# הרצת הפרויקט במצב פיתוח
npm start

# בניית הפרויקט לסביבת ייצור
npm run build
```
